# **App Name**: Patty Bros Karaoke Sign-Up

## Core Features:

- Singer Song Request Form: Provides a mobile-first interface for singers to submit required song/artist/singer details, along with optional notes, key/tempo adjustments, and conditional contact information.
- Interactive Song Catalog: Enables singers to browse and search a pre-loaded catalog (imported via CSV) and pre-fill song/artist fields on the request form with a tap confirmation.
- Host Live Dashboard: A password-protected interface for hosts to view, filter, and update incoming song requests in real-time, displaying submission details, timestamps, and current status.
- Submission Availability & Export Control: Allows hosts to toggle request submission availability (open/closed), view status counters for submitted songs, and export all submission data as a CSV file.
- AI Request Note Assistant: Provides the host with a tool to automatically generate a concise summary of any detailed notes provided by singers.
- Rolling 30-Day Data Pruning: Ensures that all submission records and associated singer contact details are automatically purged from the system 30 days after their submission.

## Style Guidelines:

- The visual design centers on a 'Sleek Professionalism' theme, leveraging a sophisticated black, white, and grey base. The primary brand accent color is a muted, deep blue-grey (#6C727F), chosen for its ability to convey modern professionalism without being overtly bright. This color adds depth and brand identity to the otherwise neutral aesthetic. The default background for the dark mode, fitting the karaoke club environment, is an extremely dark, subtly blue-tinged grey (#18191A), providing a clean, distraction-free canvas. The accent color is a vibrant sky blue (#4DD5FF), which creates effective contrast for interactive elements and highlights against the darker backdrop, offering a touch of dynamic energy.
- For light mode, the primary brand accent color (#6C727F) will serve as a prominent detail and text color, maintaining brand consistency. The background will shift to a near-white neutral, allowing the vibrant sky blue accent (#4DD5FF) to continue providing clear calls to action and emphasis.
- Both headlines and body text will utilize 'Inter', a grotesque-style sans-serif. Its modern, objective, and neutral characteristics perfectly align with the app's clean, minimalist, and distraction-free design principles, ensuring excellent readability across various screen sizes for both forms and data displays.
- Utilize a consistent set of clean, minimalist line icons. These will provide clear visual cues for notes, contact information, submission statuses, and dashboard actions, ensuring intuitive navigation and quick comprehension for both singers and hosts within the uncluttered interface.
- The singer's interface is designed mobile-first, ensuring optimal usability and aesthetic appeal on small screens with a single-column, well-spaced form layout. The host dashboard is optimized for desktop and horizontal tablets, featuring a structured, multi-column data table and intuitive control panels for efficient management.
- Subtle, fluid animations will be used to enhance user experience. These include smooth transitions for button states upon submission, clear processing indicators, and elegant real-time updates on the host dashboard as new song requests arrive or statuses change, providing responsive feedback without being distracting.
